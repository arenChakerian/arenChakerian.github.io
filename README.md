# Aren's portfolio

Three files, no frameworks:

- `index.html` — the content
- `style.css` — the look (re-theme it by editing the 5 tokens at the top)
- `game.js` — the brick-breaker in the hero

## Preview it

Double-click `index.html`. It opens in your browser. Edit a file, save, refresh. That's the whole workflow.

## Make it yours (do this before deploying)

Search the files for `TODO` — every spot that needs your attention is marked:

1. Rewrite the About section in your own voice
2. Add/remove skill tags until every one is honest
3. Push your C++ projects to GitHub, then replace the `#` placeholder links
4. Change at least one thing about the design or the game so you can talk about it

## Deploy free on GitHub Pages

1. Create a GitHub account if you don't have one
2. Create a new **public** repo named exactly `YOURUSERNAME.github.io`
3. Upload these three files (drag and drop works on github.com, or use git)
4. Wait ~2 minutes. Your site is live at `https://YOURUSERNAME.github.io`

Every future edit: change the file, commit, and the site updates itself.
